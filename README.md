# 억울함해결사

교통사고와 법률 분쟁을 겪는 사용자가 사실과 증거를 정리하고, 다음 행동을 놓치지 않으며, 변호사 상담을 준비하도록 돕는 법률 사건 준비 플랫폼이다.

## 현재 단계

제품 기획과 프론트엔드 검증 단계다. 첫 구현은 종합 법률 AI가 아니라 다음의 짧은 사용자 흐름을 검증한다.

1. 사용자가 자신의 상황을 쉬운 메뉴에서 선택한다.
2. 로그인 없이 초기 대응 체크리스트를 받는다.
3. 사건 저장·자료 업로드·AI 정리가 필요할 때 로그인한다.
4. 정리된 사건 패킷을 사용자가 직접 선택한 변호사에게 공유한다.

## 권장 첫 구현

- 홈
- `보험사 과실비율이 억울해요` 입력 마법사
- 무료 초기 체크리스트 결과
- 로그인 유도 화면
- 내 사건 대시보드 목업
- AI 사건 리포트 목업
- 변호사 목록과 프로필 목업
- 변호사·관리자 대시보드 골격

## 현재 구현

`apps/web`에 P0 프론트엔드 검증 앱이 구현되어 있다. 홈에서 상황을 선택하고, 보험사 과실비율 이견 가이드를 거쳐 무료 체크리스트와 로그인 유도까지 이어진다. AI 리포트는 Cloudflare Pages Functions를 통해 LLM에 사실 정리 초안을 요청할 수 있으며, 키는 서버 환경변수에만 둔다. 인증·파일 업로드·사건 저장은 아직 목업이다.

```bash
cd apps/web
npm install
npm run dev
```
